# Changelog

- 首个正式对外发布版本。
- Dock 化多页签界面（Sessions / Codex / Review / Tools / Settings）。
- Anthropic 与 OpenAI 提供商适配层，支持流式响应。
- 远端 MCP 客户端互通，可接入外部 MCP 服务器。
- 本地工具系统与工具目录，支持编辑器侧工具路由。
- 上下文索引服务，快速建立项目文件索引。
- 会话管理，支持多会话切换与持久化。
- 执行链与审批流，结构化管控 AI 写操作。
- 变更集服务，聚合并预览 AI 提出的修改。
